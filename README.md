## Mathmetical Formula


$\hat{\beta}=(X^TX)^{-1}X^TY$
<br/> 
$\hat{Y}=\hat{\beta}X$
<br/> 
$\hat{\epsilon}=Y-\hat{Y}$
<br/> 
$H=X(X^TX)^{-1}X^T$
$SSR=Y^T(H-1/n)Y$
$MSR=SSR/(p-1)$
$\hat{\sigma^2}=\frac{\hat{\epsilon}^T\hat{\epsilon}}/{n-p}$
$Var(\hat{\beta})=X^TX\hat{\sigma^2}$
$SE(\hat{\beta})=\sqrt{Var(\hat{\beta})}$
$F=\frac{MSR}/{\hat{\sigma^2}}$
$t=\frac{\hat{\beta}}/{SE(\hat{\beta})}$
